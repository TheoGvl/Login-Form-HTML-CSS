# Premium Neumorphic Login Form

A modern, elegant, and responsive login interface built entirely with HTML and CSS. This project showcases the **Neumorphism (Soft UI)** design trend, creating a 3D illusion where elements appear to extrude from or sink into the background using advanced CSS shadow techniques.

## Features
* **Pure CSS Design:** No JavaScript or complex frameworks required.
* **Neumorphic UI:** Soft, realistic 3D effects using layered `box-shadow` and `inset` properties.
* **Floating Background Shapes:** Geometric background elements that add depth and a premium feel to the layout.
* **Interactive States:** Smooth hover and active click effects that make the buttons and inputs feel tactile and responsive.
* **Integrated Icons:** Uses FontAwesome for a clean, professional user experience.

## Technologies Used
* **HTML5** Semantic structure
* **CSS3** Flexbox, Box-shadows, Absolute positioning
* **Google Fonts** Poppins
* **FontAwesome 6** Icons

## How to Use
1. Clone this repository or download the files.
2. Ensure both `index.html` and `style.css` are in the same folder.
3. Simply double-click `index.html` to open it in any modern web browser.

## Design Philosophy
The core concept of Neumorphism relies on the background and the elements sharing the exact same hex color (in this case, `#e6e9f0`). The structure and depth are defined purely by manipulating light and shadow, resulting in a minimalist yet highly engaging interface.
